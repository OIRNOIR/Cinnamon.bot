Cinnamon Privacy

As the developers of Cinnamon, we value your privacy.

We only collect and store:

  • User IDs, used for storing user settings, and determining the reroll winner of a giveaway.
  • Server, Channel, and Message IDs, used for storing server and channel-specific settings, and to end giveaways.
  • Emoji IDs, for storing a custom emoji in premium servers.
  • Any information you put in a giveaway, such as the prize, time, winners, requirements, etc.

All giveaway-related data are stored for a maximum of 24 hours after the giveaway ends.
All server-related data are stored until Cinnamon leaves the server.
All user-related data are stored until cleared with the removemydata command. We currently do not store user-specific data. If this changes, I will update this page.

We do not store any logs of any kind.

No messages that are received by Cinnamon are stored. If at any point Cinnamon needs to store messages, they will be encrypted using strong, secure encryption.
